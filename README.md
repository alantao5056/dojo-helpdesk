Simple todolist-style Next js app. 

## Getting Started
1. Clone the repository onto your own computer
```
git clone https://github.com/alantao5056/dojo-helpdesk.git
```
2. Install necessary dependencies
```
npm install
```
3. Start the JSON server (make sure to start it on port 4000)
```
npm run json-server
```
4. Run the development server
```
npm run dev
```

5. Enjoy!
